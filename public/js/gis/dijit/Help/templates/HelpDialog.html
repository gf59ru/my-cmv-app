Навигация по карте с помощью мыши и клавиатуры:
  • Удерживание левой кнопки мыши - перемещение карты
  • SHIFT + Левая кнопка мыши - расположить выбранную точку по центру
  • SHIFT + Удерживание кнопки мыши - приблизить карту к нарисованному квадрату
  • SHIFT + CTRL + Удерживание кнопки мыши - отдалить карту от квадрата
  • Колесо мыши - приблизить/отдать карту
  • Клавиши со стрелками - перемещение карты
  • + Приблизить карту
  • - Отдалить карту
  • Двойной щелчок Расположить точку по центру и приблизить карту
A variety of searches can be performed in the search box:
  • Search by Address
  • Search by Place Name
  • Search By Zip Code, County, etc.
In addition to Search capabilities, the following tools are provided:
  • Measure
    • The measure tool provides the capabilities to draw a point, line, or polygon on the map and specify the unit of measurement.
  • Print
    • This map can be exported to varouis formats and layouts
